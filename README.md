# 🏭 AI Production Loss Agent

An AI-powered Production Loss Explanation Agent built using Streamlit and Machine Learning.

## 🚀 Features
- Production Dashboard
- KPI Monitoring
- Production Loss Prediction
- Root Cause Analysis
- Interactive Charts
- AI Insights

## 🛠 Tech Stack
- Python
- Streamlit
- Pandas
- Scikit-learn

## ▶️ Run the Project

```bash
pip install -r requirements.txt
python -m streamlit run app.py
```

## 📂 Project Structure

- app.py
- prediction.py
- production_data.csv
- requirements.txt
